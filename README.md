# Китайский Мастер — Android 16

Готовый Android-проект для сборки APK через GitHub Actions без Android Studio.

## Важно
Загружай в GitHub именно содержимое этой папки в корень репозитория:

- `.github`
- `app`
- `build.gradle`
- `settings.gradle`
- `gradle.properties`
- `README.md`

Если загрузить папку целиком внутрь репозитория, GitHub Actions не увидит workflow.

## Сборка
Открой вкладку **Actions** → **Build APK** → **Run workflow**.

После успешной сборки APK будет в **Artifacts**.
