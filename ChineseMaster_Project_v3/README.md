# Китайский Мастер 汉字

Android-приложение для самостоятельного изучения китайского языка.

## Что внутри

- 214 ключей Канси — первый этап обучения
- Базовые иероглифы после ключей
- Пиньинь и тоны как отдельные тренировочные разделы
- Мини-тесты
- Поиск по ключам
- Прогресс и простое SRS-повторение: трудно / знаю / легко
- Сборка APK через GitHub Actions без Android Studio

## Как собрать APK

1. Создай репозиторий на GitHub.
2. Загрузи содержимое этой папки в корень репозитория.
3. Открой вкладку Actions.
4. Запусти workflow `Build Android APK`.
5. В завершённой сборке скачай artifact `ChineseMaster-debug-apk`.
6. Внутри будет `app-debug.apk`.

## Требования

Проект рассчитан на Android 6+ и targetSdk 36.
